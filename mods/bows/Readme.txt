A very easy bow api mod with arrows

Load a bow:
place a arrow left of the bow + use it

Bows / levels:
Wooded
Stone
Steel
Bronze
Mese
Diamond

Arrows / level:
Wooded
Steel
Gold
Diamond
Dig: digs the block it hits.
Fire: spawning fire on the block or object it hits
Build: placing a block from the right stack of the bow.
